# מאגר שבת חי

מאגר דיגיטלי המשמש כמרכז תוכן חי לשולחן השבת: שירים, קלפים מודפסים, פאנל תצוגה ועוד.  
[אתר חי](https://anlominus.github.io/Shabat/) · [מאגר GitHub](https://github.com/AnLoMinus/Shabat)

**גרסה נוכחית:** 0.3.0 (13 בנובמבר 2025)

## מה יש כאן?

- **דף בית אינטראקטיבי** – עמוד בית חדש (`index.html`) המציג את כל חלקי המאגר ודרכי השימוש בהם.
- **גלריית שירים לשבת** – כרטיסיות רספונסיביות הטוענות Markdown דינמי מתוך תיקיית `songs/` (ראו `album.html`).
- **חלון מודאלי** – קריאה נוחה של מילות השיר עם תמיכה מלאה בעברית, RTL ו-Markdown.
- **אנימציות עדינות** – חוויית משתמש חלקה באמצעות Bootstrap ו-CSS מודרני.
- **סטי קלפים מודפסים** – שלושה עמודים מוכנים להדפסה (A4 לרוחב, 7×10 ס״מ) להפעלת שולחן שבת.
- **פאנל שבת חי** – עמוד `panel.html` למסך מלא המציג קלפים מתחלפים, שעון שבת וחדשות חסד למסך הטלוויזיה.
- **תשתית תיעוד מלאה** – קוד התנהגות, מדריך תרומה, אבטחה, תמיכה ויומן אירועים.

## מבנה המאגר

```text
.
├── index.html          # דף בית מאגד לכל חלקי המאגר
├── home.css            # עיצוב ייעודי לדף הבית
├── album.html          # אלבום שירי שבת
├── scripts.js          # טעינת שירים ודינמיקה של הכרטיסים
├── styles.css          # עיצוב גלריית השירים והקלפים
├── panel.html          # פאנל שבת חי למסך מלא
├── panel.css           # עיצוב ייעודי לפאנל
├── panel.js            # לוגיקת תצוגת הפאנל
├── songs/              # קבצי Markdown של השירים (1–18)
├── data/
│   └── panel-content.json  # תוכן דינמי לפאנל השבת
├── docs/               # תיעוד משלים (מדריכי הדפסה, אפיון פאנל ועוד)
└── .github/            # תבניות Issues/PR ותמיכה במימון
```

## טכנולוגיות

- HTML5
- CSS3 + Bootstrap 5
- JavaScript ES2020
- Marked.js עבור פענוח Markdown

## מפת דרכים

- הרחבת תפריט האתר עם קישור לעמוד קלפים להדפסה (9 קלפים בגודל 7×10 ס״מ על A4).
- ניהול תכנים נוסף (למשל JSON/YAML) לטובת קלפים, מקורות ושיתופים.
- הוספת תמיכה במצב כהה ובנגישות (ARIA, קונטרסט).
